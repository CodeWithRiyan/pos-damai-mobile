# Mobile Development Skills (Expo & React Native)

This directory contains the mobile application for POS Damai. Use these specialized AI skills to build a high-performance, beautiful user experience.

## How to Use
Invoke these skills in the chat using the `@skill-name` syntax followed by your request.

### UI/UX & Design
- **`@frontend-design`**: Use to create distinctive, production-grade mobile interfaces with intentional aesthetics.
- **`@building-native-ui`**: Best for building universal layouts with Expo Router and NativeWind.
- **`@ui-ux-pro-max`**: Design intelligence for choosing color palettes, typography, and interactive patterns.
- **`@design-spells`**: Add micro-interactions and "magic" details to your components.

### Core Logic & State
- **`@react-native-architecture`**: Guidelines for building scalable mobile apps with Expo, offline sync, and native modules.
- **`@zustand-store-ts`**: Pattern-driven state management with Zustand and TypeScript.
- **`@native-data-fetching`**: Use for implementing React Query hooks and handling API data efficiently on mobile.
- **`@zod-validation-expert`**: Type-safe validation for data schemas and user forms.

### Cross-Platform & Mobile Security
- **`@mobile-design`**: Principles for touch interactions, platform conventions (iOS/Android), and offline behavior.
- **`@mobile-security-coder`**: Ensure your WebView and data handling practices are secure.
- **`@fixing-accessibility`**: Audit and improve the accessibility (ARIA, focus management) of your screens.

### Testing & Publishing
- **`@tdd`**: Use for reliable, test-driven feature development.
- **`@playwright-skill`**: For browser-level E2E testing if needed for web builds.
- **`@upgrading-expo`**: Use when you need to safely update Expo SDK versions.
